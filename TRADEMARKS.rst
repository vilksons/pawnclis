===========================
Trademarks Pawn Information
===========================

Pawn is a lightweight scripting language primarily known for its use in SA-MP (San Andreas Multiplayer) and other embedded applications. While "Pawn" itself is a widely recognized term in the programming community, it is essential to understand the implications of using it in third-party projects.

Trademark Considerations
========================
1. **Ownership**: The Pawn scripting language was originally created by ITB CompuPhase, and they maintain rights over its development and distribution.
2. **General Use**: The term "Pawn" has been used broadly in various projects without significant legal disputes, but it is important to respect the original creator's rights.
3. **Third-Party Extensions**: Tools and extensions incorporating "Pawn" in their name, such as `pawnclis`, should clarify that they are independent projects and not officially affiliated with ITB CompuPhase.
4. **Fair Use**: Using "Pawn" in a descriptive manner (e.g., "a command-line interface for Pawn scripts") is generally considered acceptable, as long as it does not mislead users into believing it is an official product.

Disclaimer
==========
`pawnclis` is an independent tool and is not officially affiliated with or endorsed by ITB CompuPhase. The use of "Pawn" in the project name is intended purely for descriptive purposes and does not imply any official connection to the original developers.

If there are any concerns regarding trademark usage, please refer to official trademark policies or seek legal advice if necessary.
